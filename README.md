# VUE NAIVE ADMIN

## 简介

基于 [VUE NAIVE ADMIN](https://github.com/zclzone/vue-naive-admin)

毕设 - 播客后台管理

- 专栏管理
  - 新建专栏
  - 删除专栏
  - 编辑专栏
- 专栏播客管理
  - 播客文件管理
  - 播放音频文件
- 审核
  - 专栏审核
  - 专栏播客审核
